# App Aluno React

Portabilidade do App Aluno de HTML, CSS e JavaScript puro para uma SPA em React com Vite.

## Requisitos atendidos

- Projeto criado na estrutura do Vite.
- Rotas com `react-router-dom`.
- Componentes reutilizáveis em `src/components`.
- Context API para compartilhar dados do usuário.
- Formulários controlados com validação simples.
- Renderização de listas com `.map()` e `key`.
- Consumo de API com `fetch`, `try/catch/finally`, `loading`, `erro` e `dados`.
- Estilização com `className` e CSS responsivo.

## Como executar

```bash
npm install
npm run dev
```

Depois acesse o endereço exibido no terminal.

## Fluxo para demonstrar

1. Faça login com um e-mail válido e qualquer senha.
2. Navegue entre Dashboard, Disciplinas, Tutor IA e Perfil.
3. Abra Tutor IA para demonstrar o consumo de API.
4. Teste os formulários de cadastro, recuperação e nova senha para ver as validações.
